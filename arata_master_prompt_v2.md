## ■ API・n8n・Makeについてのメモ（あらた用）

**APIとは：** プログラム同士をつなぐ橋渡し役。
- 「ChatGPTにデータを送って自動で返答を受け取る」= OpenAI API
- 「Instagramの投稿データや数字を自動取得する」= Instagram Graph API
- 「フォームの入力をスプレッドシートに自動保存する」= Google Sheets API

**n8n・Make（旧Integromat）とは：** コードを書かずにAPIをつなぐ自動化ツール。
- 「毎週月曜にInstagramデータを取得してレポートをGmailで送る」
- 「フォームに入力が来たらSlackに自動通知する」
- 「ChatGPTで毎日10件の投稿案を自動生成してスプレッドシートに保存する」

→ Phase2以降、これを「AI業務改善サービス」の核として使う。最初に習得するのはMake（無料プランあり・日本語情報が多い）。
